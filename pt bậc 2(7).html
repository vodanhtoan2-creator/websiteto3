Biện luận số nghiệm phương trình bậc 2

Bài toán

Xác định số nghiệm của phương trình ax2 + bx + c = 0 (a ≠ 0)

Cách làm

Thực hiện theo 3 bước:

  • B1: Xác định hệ số a, b, c
  • B2: Tính Δ = b2 − 4ac
  • B3: Biện luận số nghiệm

Kết luận

  • Δ < 0 → Phương trình vô nghiệm
  • Δ = 0 → Phương trình có nghiệm duy nhất
  • Δ > 0 → Phương trình có hai nghiệm phân biệt
