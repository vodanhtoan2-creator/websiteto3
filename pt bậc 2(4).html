Biện luận số nghiệm
phương trình bậc 2

ax2 + bx + c = 0
Điều kiện: a ≠ 0

Bài toán

Xác định số nghiệm của phương trình bậc hai.

Cách làm

  1. Xác định các hệ số a, b, c
  2. Tính Δ = b2 − 4ac
  3. Biện luận số nghiệm
  • Δ < 0 → Vô nghiệm
  • Δ = 0 → Nghiệm duy nhất
  • Δ > 0 → Hai nghiệm phân biệt
