Biện luận số nghiệm
Phương trình bậc 2

Bài toán: Xác định số nghiệm của phương trình

ax2 + bx + c = 0   (a ≠ 0)

Cách làm:

  1. Bước 1: Xác định các hệ số a, b, c
  2. Bước 2: Tính Δ = b2 − 4ac
  3. Bước 3: Biện luận số nghiệm
  • Δ < 0 → Phương trình vô nghiệm
  • Δ = 0 → Phương trình có nghiệm duy nhất
  • Δ > 0 → Phương trình có hai nghiệm phân biệt
